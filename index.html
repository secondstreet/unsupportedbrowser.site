What Is A “Browser”?

A “browser,” or “web browser,” is an app on your computer, phone, or tablet that lets you visit websites and use web applications.

Why Is My Browser Unsupported?

Like many apps, browsers need to be updated. New versions are released all the time. Newer versions of a browser will be safer, more secure, and have more features. Plus, many websites and web applications stop supporting browsers when they get too old.
